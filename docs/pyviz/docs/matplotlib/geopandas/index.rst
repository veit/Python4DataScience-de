GeoPandas
=========

`GeoPandas <https://geopandas.org/en/stable/>`_ erweitert :doc:`../pandas/index`
um Datentypen, die räumliche Operationen an geometrischen Typen ermöglicht,
wobei die geometrischen Operationen von `Shapely
<https://shapely.readthedocs.io/en/stable/>`_ ausgeführt werden. Für den
Datenzugriff verwendet GeoPandas `Fiona
<https://fiona.readthedocs.io/en/stable/>`_ und zum Plotten `descartes
<https://pypi.org/project/descartes/>`_ und :doc:`../index`.
